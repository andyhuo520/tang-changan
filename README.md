### 体力 + 风度系统

- **体力 stamina** 0-100，初始 100；每秒自动 +1 恢复（mini-game / 跑步消耗待加）
- 屏右上**绿色 4px 条**实时显示
- 食物给定量回复，酒/茶/酥山额外给**临时 charm 风度**（60-180 秒 buff）

### 文房灵感 → 接入诗会

持有文房物品自动累计 poetryBonus（笔 +10、墨/纸 +5、砚 +20、唐诗集 +25、全唐诗 +35）。
- bonus ≥ **15** → 诗会内可灵思**一次**（答错时获一次重答机会，且会标红错项）
- bonus ≥ **35** → 玩家直接成为"长安第一才子"（彩蛋）

### Loot 重构

旧的 REWARDS_GOOD/SMALL/CONSOL/PERFECT → 统一迁到 `LOOT_TABLES` (win/draw/lose/perfect)，全部用 itemId 引用。币奖单独用 `COIN_10` / `COIN_3` / `SILK_1` token 处理。所有路径（猜拳/猜谜/对诗）共享一致 grantReward。

## V10 · 诗酒小宴 · 飞花令 · 对对联 · 诗仙赞

> 古人云：花间一壶酒，独酌无相亲；举杯邀明月，对影成三人。

街头三玩之外再开一种 **盛唐文人雅集** —— 走到 NPC 旁 `E` 开对话，到末尾会出现紫粉色按钮 **🍷 对诗**（文士 / 仕女 / 老人 / 僧人 100%，兵卒 / 农夫 30%）。点开后随机抽 3 轮，混合两种唐风玩法：

| 玩法 | 形式 |
|------|------|
| **飞花令** | 给关键字 (花/月/风/春/江/酒/山/雪)，4 句中选**含此字**的诗 |
| **对对联** | 给上联，4 句中选下联（干扰项均为**其他名联的下联**，迷惑性强） |

**3 巡定胜负 · 奖品分级**：

| 战绩 | 级别 | 奖品池 |
|------|------|--------|
| 3/3 | **诗仙赞 ✦** | 🍶 一壶剑南春 / 📚 《全唐诗》/ 🖌 湖州狼毫 / 🟫 端州歙砚 / 🎨 吴道子真迹 / 🪪 诗仙赞墨色腰牌 |
| 2/3 | 才情不俗 | 小赏（三文钱 / 胡饼 / 打油诗…） |
| 1/3 | 勉强一巡 | 安慰（路人一笑 / 鬼脸） |
| 0/3 | 颗粒无收 | 文士调侃：「公子何不饮酒，少谈诗？」 |

题库规模：

- 飞花令：**8 个关键字 × 每字 ~6 含字句 + ~6 不含字句**，组合数 >180
- 对联：18 副名联（王勃 / 李白 / 王维 / 柳宗元 / 杜甫 / 白居易 / 王翰 / 王之涣 / 张九龄…）

对诗 UI 与街头小游戏统一 `.sg-card` 风格，但配色改为 **暗夜紫 + 桃花粉**，提示气泡用 **STKaiti 楷体 + 紫色左竖条**，呼应"花间一壶酒"的雅意。

## V9 · 街头玩耍 · 猜拳猜谜 + 随机赏赐 + 行囊系统

游戏模式中靠近任意路人，按 `E` 打开对话；在对话**最后一行**会出现两枚按钮：

| 按钮 | 玩法 |
|------|------|
| **🪨 猜拳** | 经典石剪布。你出招后，NPC 转手轮抽签 600ms 落定 → 胜 / 平 / 负 |
| **🎴 猜谜** | 唐风题库（民俗谜语 + 长安史实），四选一，对错均有奖 |

**胜负判定 → 不同奖品池**：

- **胜** → 十文开元通宝 / 西凉铜镜 / 滕王阁序拓本 / 龙井香囊 / 葡萄酒 / 东海明珠 / 织锦帕 / 酥山
- **平** → 三文铜钱 / 胡饼 / 打油诗 / 炒栗子 / 竹簪
- **负** → 路人一笑 / "再来" / 鬼脸（安慰，无价值但有趣）

NPC 角色对游戏的态度：

| 角色 | 愿意陪玩 |

## V7 · 长安行客 · RPG 模式

点击顶栏 **🎮 游戏模式** 按钮，从 4 个角色预设（**士子 / 商贾 / 仕女 / 游侠**）中挑选你的长安身份，输入名字，进入第三人称漫游：

| 操作 | 键位 |
| --- | --- |
| 移动 | `W` `A` `S` `D` 或方向键 |
| 疾走 | 长按 `Shift` |
| 与最近 NPC 交互 | `E` / 空格 / 回车 |
| 退出游戏 | `Esc` |

- **主线任务**：《初到长安》5 步剧情链：朱雀门 → 西市张掌柜 → 大雁塔题名 → 大明宫含元殿 → 名垂长安
- **场景自动联动**：与 217 个 NPC 都可对话；不同坊里的人物会按身份（守门将/商贾/学子/儒生/百姓/官员/胡商/僧侣）说出不同的市井段子
- **任务驱动**：在朱雀门附近的兵卒、西市的商贾、大雁塔下的学子会触发任务专属台词与剧情推进
- **HUD**：左上任务条 / 下方按键提示 / 顶部 toast 通知 / 底部对话框
- **镜头**：相机从俯瞰沙盘平滑过渡到主角第三人称跟随，主角头顶有金色光环 + 箭头标识

## 渲染特性

- **正交相机**（OrthographicCamera）+ frustum=64，sandbox 俯视感（游戏模式相机 zoom=1.4 + 平滑 lerp 跟随主角）
- 阴影范围 ±100（覆盖 120×160 单位地图）
- 雾默认 200-450 单位（天气切换会变窄）
- 后期：OutlinePass（拾取高亮）+ UnrealBloom（灯笼 / 烽火 / 金顶辉光）
- 性能：~1200+ 网格、217 NPC、移动端 30+ FPS
- 游戏模式：WASD 输入捕获 + 60Hz 主角位姿更新 + 屏幕空间最近 NPC 查询
